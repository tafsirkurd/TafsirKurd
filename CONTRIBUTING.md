# Contributing to Tafsir Kurd

First off, thank you for considering contributing to Tafsir Kurd! 🎉

## 🤝 How to Contribute

### Reporting Bugs
- Use GitHub Issues to report bugs
- Include detailed steps to reproduce
- Provide browser/device information
- Add screenshots if applicable

### Suggesting Features
- Check existing issues first
- Clearly describe the feature and its benefits
- Explain why it would be useful to users

### Pull Requests

1. **Fork the repository**
2. **Create a branch** (`git checkout -b feature/your-feature`)
3. **Make your changes**
   - Follow existing code style
   - Add comments for complex logic
   - Test thoroughly
4. **Commit your changes** (`git commit -m 'Add feature'`)
5. **Push to your fork** (`git push origin feature/your-feature`)
6. **Open a Pull Request**

### Code Guidelines

- Use clear, descriptive variable names
- Comment complex code sections
- Keep functions small and focused
- Follow existing file structure
- Test on multiple browsers
- Ensure offline functionality still works

### Testing Checklist

Before submitting a PR:
- [ ] Code works in Chrome, Firefox, Safari
- [ ] Mobile responsive
- [ ] Offline mode works
- [ ] No console errors
- [ ] Service worker still functions
- [ ] Performance not degraded

## 📝 Commit Messages

Use clear, descriptive commit messages:
- `feat:` for new features
- `fix:` for bug fixes
- `docs:` for documentation
- `style:` for formatting
- `refactor:` for code restructuring
- `perf:` for performance improvements
- `test:` for adding tests

Examples:
```
feat: Add Surah search functionality
fix: Correct bookmark save issue
docs: Update installation instructions
perf: Optimize image loading
```

## 🌍 Translations

We welcome translations! To add a new language:
1. Create language file in `/src/data/translations/`
2. Follow existing structure
3. Ensure all strings are translated
4. Test UI with new language

## 🐛 Found a Security Issue?

Please **do not** create a public issue. Email: security@tafsirkurd.com

## 📜 Code of Conduct

- Be respectful and inclusive
- Welcome newcomers
- Give constructive feedback
- Focus on what's best for the community

## ❓ Questions?

Feel free to ask questions in:
- GitHub Discussions
- Issues (tagged as "question")
- Email: info@tafsirkurd.com

Thank you for helping make Tafsir Kurd better! 🌟
