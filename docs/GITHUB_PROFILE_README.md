## 🌟 About

Building digital tools to make the Quran more accessible to Kurdish-speaking communities worldwide. Our mission is to provide high-quality Islamic resources in the **Badini Kurdish** dialect.

### 🎯 Current Focus

🚀 **Tafsir Kurd** - A Progressive Web App for studying the Quran in Kurdish
📱 Works offline, installable on any device
🌙 Dark mode, bookmarks, reading goals, and more

## 🚀 Featured Project

### [📖 Tafsir Kurd - Kurdish Quran Web App](https://github.com/tafsirkurd/Tafsir-Kurd)

A modern Progressive Web App for reading and studying the Quran in Kurdish (Badini dialect) with integrated Tafsir.

**Features:**
- ✅ Complete Quran with Kurdish translation and Tafsir
- ✅ Offline support - works without internet
- ✅ Reading goals and streak tracking
- ✅ Bookmarks and user profiles
- ✅ Dark mode for comfortable reading
- ✅ Responsive design - mobile, tablet, desktop
- ✅ 100/100 Lighthouse performance score

**Tech:** HTML5 • CSS3 • JavaScript • PWA • Supabase • Netlify
