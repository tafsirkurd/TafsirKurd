## ❓ FAQ Channel Message

Post this in **#❓-faq**:

```
❓ **FREQUENTLY ASKED QUESTIONS**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🌐 ABOUT TAFSIR KURD**

**Q: What is Tafsir Kurd?**
A: Tafsir Kurd is a web platform that allows you to read the Holy Quran with Kurdish Badini Tafsir (explanation). Track your progress, set goals, and complete the Quran with understanding.

**Q: Is it free?**
A: Yes! Tafsir Kurd is completely free to use.

**Q: What languages are supported?**
A: The Quran text is in Arabic, with Tafsir (explanation) in Kurdish Badini dialect. The interface supports Kurdish and English.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**📱 GETTING STARTED**

**Q: How do I create an account?**
A:
1. Visit https://tafsirkurd.com
2. Click "Sign Up" or "Login with Google"
3. Complete your profile
4. Start reading!

**Q: Can I use it on mobile?**
A: Yes! The website works on mobile browsers. A dedicated mobile app is planned for the future.

**Q: How do I track my progress?**
A: Your reading progress is automatically saved as you read. View your stats in your profile dashboard.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🎯 FEATURES**

**Q: Can I set daily goals?**
A: Yes! Go to your profile → Goals → Set your daily reading target in ayahs or minutes.

**Q: What happens when I complete the Quran?**
A: You'll receive a completion certificate, get the 🏆 role in Discord, and can track multiple completions!

**Q: Can I bookmark ayahs?**
A: Yes! Click the bookmark icon on any ayah to save it for later.

**Q: Is there audio recitation?**
A: Yes! Audio recitation is available for every surah.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🆘 SUPPORT**

**Q: I found a bug. Where do I report it?**
A: Report bugs in <#bug_reports> with details about the issue.

**Q: I have a feature suggestion. Where do I share it?**
A: Share feature requests in <#feature_requests>. We love hearing your ideas!

**Q: My account isn't working. Help?**
A: Ask for help in <#support>. Include details about the issue you're experiencing.

**Q: Can I delete my account?**
A: Yes. Go to Settings → Account → Delete Account. This cannot be undone.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**💬 DISCORD SERVER**

**Q: How do I get roles?**
A: React to messages in <#get_roles> to choose your roles.

**Q: What does each role mean?**
A: Check Server Settings → Roles to see role descriptions and colors.

**Q: How do I use bot commands?**
A: Type `/` in any channel to see available commands:
   • `/stats` - View platform statistics
   • `/users` - See recent users
   • `/activity` - Check activity reports

**Q: Can I share Quran content here?**
A: Yes! Share in <#media_share>. Keep it respectful and beneficial.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🤲 COMMUNITY**

**Q: Can I invite friends?**
A: Absolutely! Share this Discord invite and the website link.

**Q: Are there study groups?**
A: Yes! Join voice channels for group study sessions.

**Q: Can I request duas (prayers)?**
A: Yes! Use <#dua_requests> to request or share prayers.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**📧 CONTACT**

**Q: How do I contact the admin?**
A: Use the contact form on https://tafsirkurd.com or DM server admins.

**Q: Can I contribute to the project?**
A: We welcome contributions! Contact admins to discuss how you can help.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**Still have questions?** Ask in <#support>!
```

## How to Replace Channel IDs

When posting these messages, replace placeholders with actual channel IDs:

1. Enable Developer Mode in Discord (Settings → Advanced → Developer Mode)
2. Right-click any channel → Copy ID
3. Replace in messages:
   - `<#RULES_CHANNEL_ID>` → `<#actual_id_here>`
   - `<#GENERAL_CHAT_ID>` → `<#actual_id_here>`
   - etc.

Or just type `#` and select the channel - Discord will auto-format it!
